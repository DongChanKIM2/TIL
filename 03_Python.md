# 파이썬 기본 문법

## 1. 유용한 단축키

- h(elp) (단축키 모음집)
- Y(Markdown -> code)
- M(Code -> Markdown)
- 범위 주석(ctrl + /)
- vscode (범위 움직이기: alt + 위/아래)

## 2. 글꼴 설정(l과 1을 구분하자)

-  chrome > 설정 > 폰트 > 고정폭 글꼴 `consolas`

## 3. 유의사항

1. 내장함수(built_in_keyword) 변수로 사용 불가
2. 이름 지을 때 숫자가 먼저 올 수 없음

## 4. Scope
- Local > Enclosed  > Global > Built_in_functions
- Local에서 Global Scope 변수 사용하기위해 global 선언

## 5. 가변인자
- *args
- **kwargs
